The raw stimuli csv file is found within the stimuli subfolder. Within the prep subfolder, files for each category are found in their respective subfolders within stimuli_by_category. Within each category subfolder, vocab.csv contains the items within a given category, USE_embeddings.csv contains the word embeddings for each word in the list, USE_semantic_matrix_category_label.csv contains the 1 x N category label-items similarity matrix, and USE_semantic_matrix_items.csv contains the pairwise interitem similarity matrix. All functions to assemble semantic matrices and calculate average similarities are found in get_similarity.py.
